# Zepto-SQL-Data-Analysis
SQL Data Analysis project using Zepto dataset


## Project Overview
This project analyzes the Zepto grocery dataset using SQL. It includes data cleaning, exploratory data analysis (EDA), and business insights.

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## Dataset
- Zepto Products Dataset

## Analysis Performed
- Data Cleaning
- Discount Analysis
- Category-wise Analysis
- Inventory Analysis
- Price Comparison
- Top Selling Products

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE
- LIMIT

## Key Insights
- Found products with the highest discounts.
- Compared MRP and selling prices.
- Analyzed inventory by category.
- Identified premium and budget products.
